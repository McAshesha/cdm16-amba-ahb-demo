# CdM16 × AMBA AHB-Lite Demo Platform

Hands-on digital-design playground powered by a 16-bit CdM CPU and a full AMBA AHB-Lite bus.

A coursework demo for the *Digital Platforms* class — Logisim schematics plus a
CdM16 assembly program, wired together over an AHB-Lite interconnect.

Work in progress. More to come.
